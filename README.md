# Calculadora Python 🧮

Meu primeiro projeto em Python, feito aos 15 anos pra aprender lógica de programação.

### *O que essa calculadora faz*
- Soma, subtração, multiplicação, divisão e potência
- Verifica se a operação digitada existe
- Não deixa dividir por zero pra não quebrar o programa

### *Como rodar na sua máquina*
1. Instale o Python
2. Baixe o arquivo calculadora.py
3. No terminal, rode: python calculadora.py

### *O que aprendi fazendo isso*
- Usar input() pra pegar dados do usuário
- if, elif e else pra tomar decisões
- Converter texto pra número com float()
- f-strings pra deixar os prints bonitos
