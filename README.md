# Calculadora-Python
Minha primeira calculadora em Python, feita aos 15 anos.

** Oque ela faz: **

- Soma, subtração, multiplicação, divisão, potência.
- trata erro de divisão por zero.
- valida se a operação existe.

** Como rodar na sua máquina:**

- tenha Python instalado.
- baixe o arquivo `calculadora.py´.
- rode no terminal: python calculadora.py.
